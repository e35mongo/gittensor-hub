# What is Gittensor (SN74)?

Gittensor is **Bittensor Subnet 74**. It rewards miners for **real, merged pull requests** (and related issue-discovery work) on repositories listed in the subnet’s master registry.

- Miners link a GitHub identity (via PAT) to a Bittensor hotkey.
- Validators score merged contributions on incentivized repos.
- Emissions are allocated by **repository-bounded slices**, not a single global leaderboard of all PRs.

**Gittensor Hub** is an SN74 ops dashboard and contribution flywheel for this repo and related tooling. Hub Score (when live) is a **product reputation** signal — it is **not** the same as TAO emissions from validators.
